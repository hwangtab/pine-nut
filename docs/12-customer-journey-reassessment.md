## 3. 핵심 순간 (Critical Moments)

### Aha Moment — "이건 진짜구나"
- **현재**: 타임라인에서 실제 날짜와 사건을 볼 때 (텍스트만)
- **이상**: 주민의 얼굴, 숲의 사진, 벌채 현장을 **직접 볼 때**
- **격차**: 텍스트 → 시각 증거의 단절이 Aha Moment를 약화시킴

### Moment of Truth — "나도 함께할까?"
- **현재**: 서명 페이지 도달 시 (전체 방문자의 추정 15~20%만 도달)
- **문제**: 서명이 실제 저장되지 않는다는 걸 눈치채면 **신뢰 붕괴**
- **해결**: 실제 백엔드 연동 + 실시간 카운터가 최우선

### Churn Trigger — "여기까지만 볼게"
1. **첫 방문 3초**: 실제 사진 없이 텍스트+SVG만 보이면 → "일반적인 청원 사이트" 판단 후 이탈
2. **스토리 중간**: 긴 텍스트만 계속되면 → 스크롤 포기
3. **서명 후**: 실제로 저장되지 않는 걸 감지하면 → 사이트 전체 불신

## 8. 핵심 결론

### 가장 큰 이탈 지점
**첫 방문 → 감정적 연결 구간** (100명 → 37명, 63% 이탈)
- 원인: **실제 사진/영상 제로** — 텍스트만으로는 "진짜"라는 확신을 줄 수 없음

### 최소 투자 최대 효과 TOP 3
1. **서명 백엔드 연동** — 사이트 핵심 기능이 작동하지 않는 건 치명적
2. **실제 사진 10~15장 확보** — Unsplash/Pexels 풍경 + 뉴스 기사 사진 활용 가능
3. **히어로 섹션 시각적 임팩트 강화** — 사진 배경 + 카운터 애니메이션

### 기술적으로 즉시 구현 가능한 것
- Supabase 서명 테이블 + API 연동 (프론트 이미 구축됨)
- Stibee 뉴스레터 폼 연동 (임베드 코드만 삽입)
- 오픈소스 풍경/자연 사진으로 임시 비주얼 보강
- Kakao Map API로 위치 표시
