**Date**: 2026-03-10
**Context**: 제로 예산 출시 캠페인
**Constraint**: 자원봉사 기반, 마케팅 예산 없음

## Marketing Campaign Ideas

### Campaign 1: "680번의 외침" 카카오톡 릴레이

| 항목 | 내용 |
|------|------|
| **Channel** | 카카오톡 |
| **Concept** | 680여 차례 집회 중 상징적인 사진 10장을 선별하여 카드뉴스로 제작. "1번째 외침 → 100번째 → 300번째 → 680번째"로 시간의 무게를 시각화. 마지막 카드에 웹사이트 링크 + "681번째는 당신의 서명입니다" |
| **Effort** | 낮음 (사진 확보 + Canva 카드뉴스 제작) |
| **Expected Impact** | 높음 — 카카오톡은 한국 최대 바이럴 채널, 숫자의 반복이 감정적 임팩트 극대화 |
| **Timing** | 출시일 D-Day |
| **Action** | 주민회 + 환경단체 단톡방 동시 배포 → 2차 공유 유도 |

### Campaign 2: "7년 7장" 트위터(X) 스레드

| 항목 | 내용 |
|------|------|
| **Channel** | 트위터(X) |
| **Concept** | 2019~2025 각 연도별 대표 사진 1장 + 한 줄 설명으로 7개 트윗 스레드 작성. "2019년: 처음으로 마을 회의가 열렸습니다..." → "2025년: 여전히 싸우고 있습니다." 마지막 트윗에 웹사이트 링크 |
| **Effort** | 낮음 (사진 7장 + 트윗 7개) |
| **Expected Impact** | 중간~높음 — 트위터 스레드 형식은 사회 이슈 확산에 효과적, RT 체인 기대 |
| **Timing** | 출시 D+1 (출시 당일 카카오톡, 다음 날 트위터) |
| **Action** | 환경 활동가·기자 멘션/태그하여 초기 RT 확보 |

### Campaign 3: "할머니의 편지" 인스타그램 릴스

| 항목 | 내용 |
|------|------|
| **Channel** | 인스타그램 |
| **Concept** | 70대 주민 1명이 카메라를 보며 짧은 메시지를 전하는 30초~1분 영상. 편집 없이 자연스러운 톤. "저는 올해 78살입니다. 7년 동안 매주 군청 앞에 섰습니다. 제 이야기를 들어주세요." 자막 필수 |
| **Effort** | 중간 (현지 촬영 필요, 편집은 간단) |
| **Expected Impact** | 매우 높음 — 고령 주민의 직접 호소는 가장 강력한 감정적 콘텐츠. 릴스 알고리즘에 유리 |
| **Timing** | 출시 D+3 |
| **Action** | 인스타 환경 계정들 태그, #풍천리 #양수발전소반대 해시태그 |

### Campaign 4: "기자님, 이 이야기를 써주세요" 미디어 피칭

| 항목 | 내용 |
|------|------|
| **Channel** | 이메일 + 전화 (기자 직접 피칭) |
| **Concept** | 보도자료 + 프레스킷을 환경·사회 전문 기자 20명에게 개인화된 이메일로 발송. 제목: "7년 680번, 대한민국에서 가장 오래된 주민 환경 투쟁 — 웹사이트로 기록했습니다". 기사 앵글 3개 제안: ① 7년 투쟁 르포 ② 전과자가 된 할머니 인물 기사 ③ 디지털로 보존하는 시민 운동 |
| **Effort** | 중간 (보도자료 작성 + 기자 리스트 확보 + 개별 이메일) |
| **Expected Impact** | 높음 — 기사 1건이 수천 명의 신규 유입 창출. 프레스킷이 있으면 기자 작업 부담 감소 |
| **Timing** | 출시 D-Day 오전 |
| **Target 매체** | 한겨레, 경향신문, 프레시안, 한국농정신문, 오마이뉴스, 민중의소리 |

### Campaign 5: "지구의 날, 풍천리를 기억해주세요" 시즌 캠페인

| 항목 | 내용 |
|------|------|
| **Channel** | 전채널 (카카오톡, 트위터, 인스타, 뉴스레터) |
| **Concept** | 지구의 날(4/22), 세계 환경의 날(6/5), 산의 날(10/15) 등 환경 기념일에 맞춘 특별 콘텐츠 발행. "지구의 날에 가장 먼저 떠오르는 곳, 풍천리". 해당 날짜에 서명 마일스톤 챌린지 |
| **Effort** | 낮음 (기존 콘텐츠 재활용 + 기념일 프레임) |
| **Expected Impact** | 중간 — 기념일 해시태그 트렌드에 편승, 환경 관련 검색 증가 시기 활용 |
| **Timing** | 출시 후 가장 가까운 환경 기념일에 맞춰 집행 |
| **Calendar** | 3/22 세계 물의 날, 4/22 지구의 날, 6/5 세계 환경의 날, 10/15 산의 날 |

## Positioning Options

| # | Positioning | 대상 | 강점 | 리스크 |
|---|-----------|------|------|--------|
| 1 | "대한민국에서 가장 오래된 주민 환경 투쟁" | 일반 시민 | "7년"이라는 숫자의 무게감, 역사적 의미 | 더 오래된 사례가 있을 수 있음 → "가장 오래 지속된 중 하나"로 완화 |
| 2 | "전과자가 된 할머니·할아버지들의 이야기" | 감정적 공감 시민 | 강력한 감정적 후크, 불의에 대한 분노 유발 | 과도한 감정 자극 비판 가능 → 사실에 기반한 톤 유지 |
| 3 | "잣나무 숲이 사라지기 전에" | 환경 감수성 높은 시민 | 자연/생태계 보호 프레임, 시각적 소구력 | 주민 생존권보다 환경이 앞서는 인상 → 병행 사용 |

**Recommended**: #1을 메인 포지셔닝으로, #2를 감정적 훅으로, #3을 환경 프레임으로 상황별 사용

## Value Prop Copy

### Tagline
> **7년, 680번의 외침. 풍천리의 이야기를 들어주세요.**

### Elevator Pitch (2~3문장)
> 강원도 홍천 풍천리의 70대 주민들은 양수발전소 건설에 반대하며 7년간 680번이 넘게 거리에 섰습니다. 잣나무 숲과 마을 공동체를 지키려는 이 싸움에, 당신의 이름을 보태주세요.

### Landing Page Hero
> **Headline**: 7년간 680번, 매주 거리에 선 할머니·할아버지들
> **Subheading**: 풍천리 양수발전소 건설을 멈춰주세요. 당신의 서명이 힘이 됩니다.

### Sales One-liner (연대 요청 시)
> 풍천리는 제2의 밀양입니다. 70대 주민들의 7년 투쟁에 함께해주세요.

### KakaoTalk Share Text
> 7년 동안 매주 싸우고 있는 풍천리 할머니·할아버지들. 서명 하나가 큰 힘이 됩니다 🙏
