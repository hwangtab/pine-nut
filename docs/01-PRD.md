# Product Requirements Document: 풍천리 양수발전소 반대 투쟁 알림 웹사이트

**Author**: 풍천리 주민 연대
**Date**: 2026-03-10
**Status**: Draft
**Stakeholders**: 풍천리 주민회, 환경단체 연대, 자원봉사 개발팀

---

## 1. Executive Summary

강원도 홍천군 화촌면 풍천리 주민들의 7년간 양수발전소 반대 투쟁을 기록하고 알리기 위한 웹사이트를 구축합니다. 70대 이상 고령 주민들이 주축이 된 680여 차례의 민주적 저항 활동을 아카이빙하고, 일반 시민·환경단체·언론·정책결정자들에게 생태계 파괴와 생존권 위기의 실상을 전달하여 연대와 지지를 이끌어내는 것이 목적입니다. 프로젝트 백지화라는 명확한 목표 달성을 위해 여론 형성과 실질적 지원(서명, 후원, 자원봉사)을 연결하는 플랫폼이 필요합니다.

---

## 2. Background & Context

### 문제 배경
- 2019년부터 풍천리 주민들은 양수발전소 건설 계획에 만장일치로 반대 의사를 표명
- 양수발전소 건설 시 예상 피해: 대규모 생태계 파괴, 소음·분진 피해, 잣 생산지(주요 생계수단) 소멸, 지역 공동체 와해
- 7년간 매주 홍천군청 앞 집회, 상경 투쟁 등 680여 차례 반대 활동 전개
- 일부 주민은 투쟁 과정에서 전과자가 되는 피해를 입음
- 고령 주민 주축의 투쟁이라 디지털 홍보 역량 부족 → 외부에 잘 알려지지 않음

### 왜 지금 웹사이트가 필요한가
- 7년간 축적된 투쟁 기록이 체계적으로 아카이빙되지 않아 유실 위험
- SNS 단편 게시물만으로는 사안의 전체 맥락 전달이 어려움
- 전국적 환경 연대 움직임 속에서 풍천리 사례를 가시화할 필요
- 고령 주민들의 목소리를 디지털로 보존하고 증폭할 영구적 거점이 필요

### 선행 사례(Prior Art)
- 밀양 송전탑 반대 투쟁 웹사이트 / 다큐멘터리
- 제주 강정마을 해군기지 반대 웹사이트
- 설악산 케이블카 반대 캠페인 사이트
- 국제: Standing Rock Sioux 파이프라인 반대 캠페인 사이트

## 3. Objectives & Success Metrics

### Goals (성공의 모습)
1. 풍천리 투쟁의 전체 맥락을 누구나 이해할 수 있도록 체계적으로 전달
2. 온라인 서명·후원·자원봉사 등 실질적 연대 행동을 유도
3. 언론·정책결정자가 참조할 수 있는 공신력 있는 정보 허브 구축
4. 주민들의 기록과 목소리를 디지털로 영구 보존

### Non-Goals (범위 밖)
1. 실시간 라이브 스트리밍 플랫폼 구축 — YouTube/Instagram Live 연동으로 대체
2. 법률 자문 서비스 직접 제공 — 법률 지원 단체 연결 링크로 대체
3. 전자상거래(잣 판매 등) 기능 — Phase 1에서는 제외, 추후 검토
4. 다국어 전체 번역 — Phase 1에서는 영문 요약 페이지만 제공

| 언론 기사 내 웹사이트 인용 | 0 | 월 3건+ | 미디어 모니터링 |

| 뉴스레터 구독자 수 | 0 | 2,000명+ | 메일 서비스 |

## 6. Solution Overview

### 기술 스택 (권장)
- **프레임워크**: Next.js (SSR/SSG로 SEO 최적화, 빠른 로딩)
- **CMS**: Strapi 또는 Sanity (비개발자 콘텐츠 관리)
- **호스팅**: Vercel 또는 Cloudflare Pages (무료/저비용, 빠른 CDN)
- **DB**: PostgreSQL (서명 데이터 등) — Supabase 활용 시 무료 티어 가능
- **결제**: 토스페이먼츠 or 카카오페이 연동 (후원금)
- **메일**: Stibee (한국형 뉴스레터 서비스, 무료 티어)
- **분석**: Google Analytics 4 + Plausible (프라이버시 친화적 대안)

### 핵심 설계 원칙
1. **스토리 퍼스트**: 데이터보다 사람의 이야기가 먼저 — 랜딩에서 주민 얼굴과 목소리로 시작
2. **접근성 우선**: 큰 글씨(기본 18px+), 고대비, 단순한 네비게이션, 고령자 테스트
3. **모바일 퍼스트**: 카카오톡 공유 유입이 주요 경로이므로 모바일 최적화 필수
4. **저비용 유지**: 오픈소스 + 무료 티어 최대 활용, 월 유지비 5만원 이내 목표
5. **SEO 최적화**: "풍천리 양수발전소", "홍천 양수발전소 반대" 등 핵심 키워드 타겟

### 사이트맵 (Phase 1)

```
홈 (스크롤 스토리텔링 랜딩)
├── 풍천리 이야기 (배경 + 타임라인)
├── 왜 반대하는가 (환경·생존권 쟁점 정리)
├── 주민의 목소리 (인터뷰·사진 갤러리)
├── 소식 (뉴스·공지 게시판)
├── 함께해주세요
│   ├── 서명하기
│   ├── 후원하기
│   └── 공유하기
├── 자료실 (프레스킷·보도자료·데이터)
└── English Summary
```

## 8. Timeline & Phasing

### Phase 1: 핵심 사이트 구축 (4~6주)

| 주차 | 마일스톤 |
|------|---------|
| 1주 | 콘텐츠 수집(사진·영상·인터뷰), 도메인 확보, 디자인 시안 |
| 2주 | 와이어프레임 확정, 주민 피드백 반영, 개발환경 세팅 |
| 3~4주 | 프론트엔드 개발(랜딩, 타임라인, 서명, 뉴스), CMS 연동 |
| 5주 | 후원 결제 연동, SEO 설정, 모바일 QA |
| 6주 | 주민·활동가 베타 테스트, 수정, **출시** |

### Phase 2: 확장 (출시 후 1~2개월)
- 영문 페이지, 뉴스레터, 캘린더, 환경 데이터 페이지
- CMS 관리자 교육

### Phase 3: 고도화 (출시 후 3~6개월)
- 드론 영상/360도 갤러리, 자원봉사 시스템, 연대 네트워크 맵
- 트래픽 분석 기반 UX 개선

### 의존성
- 콘텐츠(사진·영상·인터뷰 원본)가 Phase 1 개발 2주차 전까지 확보되어야 함
- 후원 결제 연동은 주민회 계좌/법인 확정 후 진행 가능
- 개인정보 처리방침은 서명 기능 출시 전 법률 검토 완료 필요
