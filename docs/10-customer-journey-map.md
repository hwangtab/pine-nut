## Primary Journey: 일반 시민

### Stage 1: Discovery (발견)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 카카오톡 (지인 공유), 트위터/인스타 (환경단체 게시물), 뉴스 기사 링크 |
| **User Action** | 카카오톡에서 지인이 공유한 링크 또는 카드뉴스를 보고 클릭 |
| **생각** | "풍천리? 어디지?", "양수발전소가 뭐지?", "7년이나 싸웠다고?" |
| **감정** | 호기심 😮 → 가벼운 관심 |
| **Pain Points** | 링크 미리보기가 불분명하면 클릭하지 않음. "또 서명해달라는 거 아니야?" 피로감 |
| **Opportunities** | 카카오톡 OG 이미지에 "7년, 680번, 70대 할머니의 싸움" 핵심 숫자 노출 → 클릭 유도 |
| **KPI** | 카카오톡 공유 → 클릭 전환률 (CTR) |

### Stage 2: First Visit (첫 방문)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 웹사이트 랜딩 페이지 (모바일) |
| **User Action** | 히어로 이미지와 숫자 확인, 스크롤 시작 |
| **생각** | "이 사람들 진짜 오래 싸웠네", "사진이 인상적이다" |
| **감정** | 호기심 😮 → 놀라움 😲 |
| **Pain Points** | 로딩 느리면 이탈 (3초 룰). 첫 화면에서 맥락이 안 잡히면 이탈. 글이 너무 길면 이탈 |
| **Opportunities** | 3초 안에 핵심 메시지 전달. 히어로 이미지에 주민 얼굴 — 인간적 연결. 로딩 최적화(next/image, ISR) |
| **KPI** | 이탈률 (Bounce Rate), 첫 스크롤 비율 |

### Stage 3: Understanding (이해)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 스크롤 스토리텔링 본문, 타임라인 페이지 |
| **User Action** | 스토리를 스크롤하며 읽음. 일부는 타임라인 페이지로 이동하여 상세 탐색 |
| **생각** | "잣나무 숲이 다 없어진다고?", "할머니들이 전과자가 됐다고?", "이건 좀 심하네" |
| **감정** | 놀라움 😲 → 분노 😠 → 안타까움 😢 |
| **Pain Points** | 양수발전소 개념을 모르면 이해 장벽. 텍스트만 길면 집중력 저하. 타임라인이 너무 상세하면 압도감 |
| **Opportunities** | "양수발전소란?" 간단 인포그래픽 삽입. 사진·영상으로 시각적 전환 자주. 타임라인은 핵심 15개 이벤트만 기본 표시, "더보기"로 확장 |
| **KPI** | 스토리 완독률 (스크롤 90%+), 타임라인 페이지 진입률, 평균 체류 시간 |

---

### Stage 4: Emotional Response (감정적 연결)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 주민 인터뷰 섹션, 주민 사진, 집회 사진 |
| **User Action** | 주민의 실제 말을 읽거나 영상을 시청 |
| **생각** | "우리 할머니·할아버지 같은 분들이...", "이분들을 도울 수 있는 방법이 있을까?" |
| **감정** | 안타까움 😢 → 공감 💛 → 행동 의지 💪 |
| **Pain Points** | 주민 인터뷰가 없으면 감정적 연결 약함. 사진 품질이 낮으면 임팩트 감소 |
| **Opportunities** | **Aha Moment**: "전과자가 된 할머니" 스토리가 가장 강력한 감정적 트리거. 주민 실명+사진+직접 인용구 조합이 핵심. Phase 2 인터뷰 영상은 30초 숏폼으로 |
| **KPI** | 인터뷰 섹션 도달률, 영상 재생률 (Phase 2) |

> 💡 **Aha Moment**: "70대 할머니·할아버지가 7년간 매주 싸워왔고, 그 과정에서 전과자가 되었다"는 사실을 인지하는 순간

---

### Stage 5: Action (행동 전환)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 서명 페이지, 후원 페이지, 공유 버튼 |
| **User Action** | 서명 폼 작성 → 제출 / 카카오톡 공유 버튼 클릭 / 후원 계좌 확인 |
| **생각** | "서명이라도 해야지", "이거 친구한테도 알려줘야겠다", "후원은... 다음에" |
| **감정** | 행동 의지 💪 → (서명 완료 시) 만족감 😊 |
| **Pain Points** | 서명 폼이 복잡하면 이탈 (이름+이메일 외 추가 필드 금지). 개인정보 불안감. 후원은 심리적 장벽이 높음 (Phase 1에서는 계좌이체만) |
| **Opportunities** | 서명 폼은 최소 필드(이름, 이메일, 동의). "N번째 서명자" 표시로 참여 동기 부여. 서명 후 즉시 공유 유도 ("서명했어요" 카카오톡 공유 템플릿). 후원은 낮은 기대치 설정 ("커피 한 잔 값") |
| **KPI** | 서명 전환률, 공유 전환률, 후원 페이지 조회수 |

> 💡 **Moment of Truth**: 서명 폼 제출 버튼을 누르는 순간. 이 전환을 최대화하는 것이 핵심.

---

### Stage 6: Post-Action (행동 후)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 감사 페이지/메시지, 실시간 서명 카운터 |
| **User Action** | 감사 메시지 확인, 전체 서명 수 확인, 다른 응원 메시지 읽기 |
| **생각** | "나도 함께한 사람 중 하나", "이만큼 많은 사람이 관심을 갖고 있구나" |
| **감정** | 만족감 😊 → 연대감 🤝 |
| **Pain Points** | 감사 메시지가 없으면 허탈함. "그래서 이 서명이 어디에 쓰이지?" 불안 |
| **Opportunities** | 개인화된 감사 메시지 ("지현님, 2,847번째로 함께해주셨습니다"). 서명이 어떻게 쓰이는지 안내 ("모아진 서명은 국회와 정부에 전달됩니다"). 즉시 공유 CTA ("친구에게도 알려주세요") |
| **KPI** | 서명 후 공유 전환률, 감사 페이지 체류 시간 |

### Stage 7: Ongoing Engagement (지속 참여)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 뉴스레터, 소식 페이지, SNS |
| **User Action** | 뉴스레터 구독, 간헐적 재방문, SNS 팔로우 |
| **생각** | "그 뒤에 어떻게 됐지?", "아직도 싸우고 있구나" |
| **감정** | 지속적 관심 🧐 → (업데이트 시) 감정 재환기 |
| **Pain Points** | 업데이트가 없으면 잊혀짐. 뉴스레터가 너무 자주 오면 구독 해지. 재방문 동기 부족 |
| **Opportunities** | 월 1~2회 뉴스레터 (최적 빈도). 서명 마일스톤 알림 ("5,000명 돌파!"). 새로운 소식 알림. 집회/행사 일정 공유 |
| **KPI** | 뉴스레터 개봉률, 재방문 비율, 구독 해지율 |

### Stage 8: Advocacy (옹호/확산)

| 항목 | 내용 |
|------|------|
| **Touchpoints** | 개인 SNS, 카카오톡 단톡방, 오프라인 대화 |
| **User Action** | 지인에게 적극적으로 알림, 블로그/SNS에 글 작성, 오프라인 집회 참여 |
| **생각** | "이건 더 많은 사람이 알아야 해", "나도 뭔가 더 할 수 있을까" |
| **감정** | 사명감 🔥 → 연대자로서의 정체성 |
| **Pain Points** | 공유할 만한 새로운 콘텐츠가 부족. 오프라인 참여는 지리적 제약 |
| **Opportunities** | 정기적 카드뉴스/숏폼 콘텐츠 제공 (공유 용이). "서명 랭킹" 또는 "함께한 사람들" 페이지로 소속감. 자원봉사 기회 안내 (Phase 3) |
| **KPI** | 2차 공유율 (서명자의 공유 비율), UGC 콘텐츠 수 |

## Prioritized Improvements

### Quick Wins (즉시 반영)
1. **카카오톡 OG 이미지에 "7년 680번" 숫자 크게** — 클릭률 직접 영향
2. **서명 폼 필드 최소화** (이름 + 이메일 + 동의만) — 전환률 직접 영향
3. **서명 후 "N번째 서명" + 카카오톡 공유 버튼 즉시 표시** — 2차 바이럴 루프

### Medium Investment (Phase 1 내)
4. **스토리 완독률 추적 + 이탈 지점 분석** — GA4 스크롤 이벤트 설정
5. **프레스킷 원스톱 다운로드** — 언론 보도 빈도 향상

### Deep Investment (Phase 2+)
6. **주민 인터뷰 30초 숏폼 영상** — 감정적 연결의 가장 강력한 도구
7. **뉴스레터 자동화 + 세그먼트** — 시민/기자/활동가 맞춤 콘텐츠
