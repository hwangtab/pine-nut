# North Star Metric: 풍천리 웹사이트

**Date**: 2026-03-10

---

## Step 1: Business Game Classification

풍천리 웹사이트는 전통적인 SaaS/마켓플레이스가 아닌 **시민 캠페인 플랫폼**입니다.

가장 가까운 분류: **Attention Game의 변형 → "Solidarity Game"**

| Game | 적합도 | 이유 |
|------|--------|------|
| Attention Game | △ 부분 적합 | 체류 시간이 중요하지만, 단순 소비가 아닌 "이해 + 행동"이 목적 |
| Transaction Game | △ 부분 적합 | 서명·후원이라는 거래가 있지만, 반복 거래보다 1회 전환이 핵심 |
| Productivity Game | ❌ 부적합 | 효율성 향상이 목적이 아님 |
| **Solidarity Game** (커스텀) | ✅ 최적합 | 가치는 방문자가 **이슈를 이해하고 행동으로 전환**할 때 생성됨 |

**핵심 가치 생성 순간**: 방문자가 풍천리 이야기를 읽고 → 감정적으로 연결되어 → 서명/후원/공유 중 하나 이상의 행동을 취하는 순간

---

## Step 2: North Star Metric

### 🌟 North Star Metric: **주간 연대 행동 수 (Weekly Solidarity Actions)**

**정의**: 1주일 동안 발생한 의미 있는 연대 행동의 총 횟수

```
Weekly Solidarity Actions = 서명 수 + 후원 건수 + SNS 공유 수 + 뉴스레터 구독 수
```

### 7가지 기준 검증

| 기준 | 충족 여부 | 설명 |
|------|----------|------|
| 1. 이해하기 쉬움 | ✅ | "이번 주 몇 명이 행동했는가" — 누구나 직관적으로 이해 |
| 2. 고객 중심 | ✅ | 방문자가 실제로 연대에 참여한 행동을 측정. 내부 지표가 아님 |
| 3. 지속적 가치 | ✅ | 매주 새로운 행동이 발생해야 하므로 일회성이 아닌 지속적 모멘텀 반영 |
| 4. 비전 정렬 | ✅ | "풍천리 투쟁을 알리고 연대를 이끌어낸다"는 비전과 직접 연결 |
| 5. 정량적 | ✅ | 숫자로 명확히 측정 가능. Supabase + GA4로 자동 집계 |
| 6. 실행 가능 | ✅ | 콘텐츠 개선, UX 최적화, SNS 캠페인 등으로 직접 영향 가능 |
| 7. 선행 지표 | ✅ | 연대 행동 증가 → 여론 형성 → 정치적 압력 → 백지화(궁극 목표)의 선행 지표 |

### 왜 다른 후보를 선택하지 않았는가

| 후보 메트릭 | 탈락 이유 |
|------------|----------|
| 누적 서명 수 | 항상 증가하는 단조 증가 지표라 "이번 주의 모멘텀"을 반영하지 못함 |
| 월간 순 방문자 | 방문만 하고 행동하지 않는 사람 포함 — 가치 전달을 반영하지 못함 |
| 콘텐츠 공유율 | 공유는 중요하지만 서명·후원을 배제하면 연대의 전체 그림을 놓침 |

## Step 3: Input Metrics (메트릭 별자리)

```
                    ┌─────────────────────────────┐
                    │   🌟 Weekly Solidarity       │
                    │      Actions                 │
                    └──────────┬──────────────────┘
                               │
            ┌──────────┬───────┼───────┬──────────┐
            │          │       │       │          │
       ┌────▼───┐ ┌───▼───┐ ┌▼────┐ ┌▼───────┐ ┌▼────────┐
       │ 신규   │ │스토리 │ │서명 │ │공유    │ │재방문   │
       │ 방문자 │ │완독률 │ │전환 │ │전환률  │ │비율    │
       │ 수     │ │       │ │률   │ │        │ │        │
       └────────┘ └───────┘ └─────┘ └────────┘ └────────┘
```

### Input Metric 1: 주간 신규 방문자 수 (Weekly New Visitors)

| 항목 | 내용 |
|------|------|
| **정의** | 1주일간 처음 방문한 고유 방문자 수 |
| **측정 도구** | Google Analytics 4 |
| **Target** | 출시 후 30일: 주 1,000명 / 90일: 주 2,500명 |
| **영향 레버** | SNS 공유 캠페인, 보도자료 배포, SEO 최적화, 환경단체 연대 공유 |
| **NSM 연결** | 유입 증가 → 행동 가능 모수 증가 → 연대 행동 증가 |

### Input Metric 2: 스토리 완독률 (Story Completion Rate)

| 항목 | 내용 |
|------|------|
| **정의** | 랜딩 페이지 스토리텔링의 마지막 섹션(CTA)까지 스크롤한 방문자 비율 |
| **측정 도구** | GA4 스크롤 이벤트 (scroll_depth 90%+) |
| **Target** | 40% 이상 (일반 랜딩 평균 20~30%) |
| **영향 레버** | 스토리텔링 품질, 로딩 속도, 시각 자료 배치, 섹션 길이 조절 |
| **NSM 연결** | 스토리를 끝까지 읽어야 감정적 연결 → 행동 전환 가능성 높아짐 |

### Input Metric 3: 서명 전환률 (Petition Conversion Rate)

| 항목 | 내용 |
|------|------|
| **정의** | 서명 페이지 방문자 중 실제 서명을 완료한 비율 |
| **측정 도구** | Supabase 서명 수 / GA4 서명 페이지 조회수 |
| **Target** | 15% 이상 (캠페인 서명 평균 10~15%) |
| **영향 레버** | 서명 폼 간소화, CTA 문구 A/B 테스트, 실시간 서명 수 표시, 응원 메시지 사회적 증거 |
| **NSM 연결** | 서명은 가장 핵심적인 연대 행동이자 정치적 압력의 기반 |

### Input Metric 4: SNS 공유 전환률 (Share Conversion Rate)

| 항목 | 내용 |
|------|------|
| **정의** | 전체 방문자 중 SNS 공유 버튼을 클릭한 비율 |
| **측정 도구** | GA4 커스텀 이벤트 (share_click, 채널별 구분) |
| **Target** | 5% 이상 |
| **영향 레버** | 카카오톡 공유 UX 최적화, 공유 시 미리보기 이미지 품질, 공유 문구 최적화, 공유 버튼 위치 |
| **NSM 연결** | 공유 → 신규 방문자 유입 → 자기 강화 성장 루프 |

### Input Metric 5: 주간 재방문 비율 (Weekly Return Rate)

| 항목 | 내용 |
|------|------|
| **정의** | 이전 주에 방문한 사용자 중 이번 주에 다시 방문한 비율 |
| **측정 도구** | GA4 코호트 분석 |
| **Target** | 10% 이상 |
| **영향 레버** | 뉴스레터 발송, 새로운 소식 업데이트 빈도, 이벤트/집회 일정 공지 |
| **NSM 연결** | 재방문자는 추가 행동(2차 공유, 후원) 가능성이 높음. 지속적 관심 = 지속적 압력 |

## Growth Loop

```
방문자 유입 → 스토리 읽기 → 감정적 연결 → 서명/공유/후원
     ↑                                          │
     │                                          ↓
     └──── 카카오톡 공유 ← 새로운 연대 행동 ←───┘
```

이 루프의 핵심 레버는 **공유 전환률**입니다. 공유가 늘면 신규 방문자가 유입되고, 그중 일부가 다시 공유하는 자기 강화 루프가 작동합니다.
