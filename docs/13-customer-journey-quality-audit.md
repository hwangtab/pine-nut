## 3. 핵심 순간 (Critical Moments) 분석

### 🔥 Aha Moment — "이건 진짜 심각한 일이구나"
- **이상적 트리거**: 풍천리의 실제 사진(잣나무숲 파괴 현장, 어르신들의 투쟁, 벌채 현장)을 보는 순간
- **현재 상태**: 스톡/AI 이미지로 인해 Aha Moment가 **발생하지 않거나 크게 약화됨**
- **격차 원인**: 실제 콘텐츠(사진, 영상) 수급 문제 → 이것은 코드가 아니라 콘텐츠 문제

### ⚡ Moment of Truth — "나도 함께할까?"
- **현재 트리거**: 랜딩 페이지 하단 CTA → 서명 페이지 도달
- **문제**: 히어로에서 감정적 몰입이 안 되면 CTA까지 스크롤하지 않음
- **핵심**: 투명 헤더 + 감성 폰트 + 실제 사진 = 첫 3초에 감정적 연결이 **선행**되어야 함

### 🚪 Churn Triggers — 이탈 지점 TOP 5
1. **히어로 3초** (불투명 헤더 + 고딕체 + 스톡 사진 → "그저 그런 청원 사이트")
2. **위협 카드** (AI 이미지 맥락 불일치 → "대충 만들었네" 인식)
3. **갤러리** (이미지 0개 → "미완성 사이트" → 신뢰 하락)
4. **카드뉴스** (잘림 + 낮은 품질 → 공유 포기 → 바이럴 차단)
5. **자료실** (보도 키트 0개 → 기자 이탈 → 미디어 노출 기회 상실)

## 6. PM 추가 발견 — 운영자 피드백 외

코드 감사와 라이브 사이트 교차 검증에서 추가로 발견한 개선 기회:

### A. 감정적 서사 아크(Narrative Arc) 부재
- **문제**: 랜딩 페이지가 "정보 전달" 구조 — 소개 → 위협 → 증언 → CTA
- **개선**: "이야기형 스크롤" 구조로 전환 — 평화로운 마을(이상) → 위협 등장(갈등) → 투쟁(절정) → 당신의 참여(해결) 4막 구조
- **근거**: 환경 캠페인 사이트 중 가장 높은 전환율을 보이는 곳들은 스크롤 기반 서사 아크를 사용 (Standing Rock, Save the Arctic 등)

### B. 모바일 퍼스트 터치 타겟 검증 필요
- **현재**: min-height: 44px 설정은 있으나, 위협 카드 영역에서 텍스트가 작아질 수 있음
- **개선**: 모바일에서 카드 내 텍스트 최소 16px 보장, 터치 영역 48px로 상향

### C. 감정 곡선 설계 (Emotional Pacing)
- **문제**: 위협 카드 4개가 연속으로 부정적 감정만 쌓음 → 과부하로 이탈
- **개선**: 위협 2개 → 희망/연대 1개 → 위협 2개 → CTA 패턴으로 감정 완급 조절

### D. 서명 페이지 진입 마찰
- **현재**: 랜딩 → 서명 페이지 이동 (페이지 전환)
- **개선**: 랜딩 페이지 내 간소화 서명 폼 인라인 삽입 (이름+이메일만) → 전환율 2배 이상 기대

### E. 소셜 프루프 부재
- **현재**: 서명 수가 서명 페이지에서만 보임
- **개선**: 랜딩 히어로 또는 CTA 섹션에 실시간 서명 수 + "방금 OOO님이 서명했습니다" 토스트 알림

### F. 페이지 간 감정 연결 끊김
- **현재**: 각 페이지가 독립적, 이전 페이지의 감정이 다음 페이지로 전달되지 않음
- **개선**: 각 페이지 상단에 핵심 인용구나 감정적 후크를 배치하여 서사 연속성 유지

### G. 404/빈 콘텐츠 처리
- **현재**: 갤러리에 "사진 준비 중" 18개 → 미완성 사이트 인상
- **개선**: 콘텐츠 없는 페이지는 숨기거나, "곧 공개됩니다" + 뉴스레터 구독 유도로 전환 (또는 메뉴에서 임시 제거)

### H. 후원 페이지 전환 최적화
- **현재**: 빠띠 캠페인 외부 링크만 → 이탈 높음
- **개선**: 계좌번호 복사 버튼 + 토스/카카오페이 송금 딥링크 → 3초 내 후원 완료 가능하게

```
               → 클래식한 한글 세리프, 격조 있는 느낌
본문:           프리텐다드 (Pretendard) — 현재 유지
               → 가독성 최적화된 산스세리프
```

### 색상 팔레트 미세 조정
- 현재 팔레트는 양호하나, 히어로 오버레이를 **녹색 그라데이션 → 짙은 남색~녹색 그라데이션**으로 변경
- 감정적 무게감을 더하면서 자연의 깊이를 표현

### 투명 헤더 동작 규격
```
스크롤 0px:     bg-transparent, 텍스트 white, 로고 white
스크롤 50px:    bg-white/95 backdrop-blur, 텍스트 dark, 로고 dark
전환:           transition 300ms ease
히어로 구간:    position absolute (히어로 위에 떠있음)
비히어로 페이지: 기존 불투명 헤더 유지
```

## 10. 핵심 결론

### 가장 치명적인 3가지 문제
1. **불투명 헤더 + 고딕체 = 첫 3초 감정 몰입 실패** → 전체 퍼널의 30% 이탈 원인
2. **AI/스톡 이미지의 맥락 불일치 = 진정성 훼손** → "대충 만든 사이트" 인식
3. **공유 콘텐츠(카드뉴스) 품질 미달 = 바이럴 차단** → 옹호 단계 65% 이탈

### PM 관점 추가 핵심
4. **갤러리/자료실이 "준비 중"인 채 메뉴에 노출 = 미완성 인상** → 없는 게 나음
5. **랜딩 → 서명 페이지 전환이 별도 페이지 = 마찰** → 인라인 서명 폼이 2배 효과적
6. **감정 곡선 미설계 = 부정적 감정 과부하** → 희망 섹션 삽입으로 완급 조절 필요
